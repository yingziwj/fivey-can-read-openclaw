先讲人话版

🧭 openclaw configure 像“牵着你逛一圈新房的带路老师”

如果 config 是自己拿镊子改总账本，configure 就是另一种路子：不让你盯着一长串路径，而是边走边问，边问边配。

你可以把它想成装修向导。它会问你门装哪、钥匙放哪、脑子用哪颗、消息从哪扇门进来。

适合第一次搭、或者想按步骤走的人 这页最重要的是记住：它是按区域带你走，不是一次把所有细节硬摊开 查看官方原文

第一眼

🏠 它最像“按房间装修”，而不是“先给你一厚本参数表”

官方这一页真正好用的地方，就是 --section。意思很简单：今天只修某几个房间，不必全屋重装。

🚪

openclaw configure

像请带路老师从大门开始，整套房子带你逛一遍。

🌐

--section web

今天只去“搜索和上网工具房”，别的房间先不碰。

🧠

--section model

今天只挑脑子和模型货架。

📮

--section channels

今天只装消息门、收信口和门牌名单。

命令翻译

🎒 这三条示例命令，其实像三种逛法

官方给的例子不多，但已经很够用了。每条都在表达一种不同的节奏。

openclaw configure

像说：“老师，我今天有空，整套房你带我慢慢走一遍。”

openclaw configure --section web

像说：“我今天只想把侦察室装好。搜索供应商、钥匙、联网这几样处理完就行。”

openclaw configure --section model --section channels

像说：“我们今天只逛两间房。一间挑脑子，一间装消息门。别的房间以后再说。”

🎈 记一句

--section 不是高级花活，它只是“今天先修哪几间房”。

大方向

🌉 Gateway 跑在哪，是这栋楼最该先定的朝向

官方特地提醒：你在向导里选 Gateway 放哪，会直接改 gateway.mode。这不是边角料，是总方向。

🏡 放本地

像总机房就设在你自己家里。第一次装机，很多人会先走这条。

🌍 放远端

像总机房在别的地方，你这里更像控制台和遥控器。

只定朝向也可以

官方明说，你甚至可以只把这个方向定下来，然后先收工，不必一次配完所有东西。

📌 一句话

先决定机房放哪，后面的钥匙、门、桥，才不会装到一半又返工。

模型那间房

🧠 agents.defaults.models 不是术语，它其实就是“货架上摆哪些脑子”

官方说模型区现在会带一个多选 allowlist。翻成人话，就是你能控制哪些模型会出现在 /model 和选择器里。

🪑

默认模型

像大厅里平时坐班的那颗主脑。

🗃

模型 allowlist

像决定还有哪些备选脑子被摆上货架，让你以后切换时看得见。

🎛

/model 与 picker

它们像前台按钮面板。货架没摆上去，按钮里自然也不会出现。

📌

一句话

这一步不是背模型名，而是在整理“平时能选哪些脑子”。

搜索那间房

🔎 --section web 像“去给它装一个侦察室”

官方这里有个很贴心的点：如果你选的是 Grok，向导还可能多追问一小步，帮你决定要不要再开 x_search 那扇侧门。

选择搜索供应商

像决定雇哪一位外勤侦察员出去找资料。

配置凭证

像把那位侦察员的工作证和通行卡配好。

Grok 额外一步

像老师又问一句：“既然你选了这位侦察员，要不要顺手把旁边那台专用望远镜也一起装上？”

🎈 一句话

Web 区不是单纯填 API Key，而是在给 OpenClaw 装一整间找资料的小工作室。

钥匙和安装

🔐 官方这里特别认真，因为“钥匙藏起来”和“服务长期值班”要一起讲清楚

这部分如果只翻术语会很干。换成场景就清楚了：钥匙可以放进秘密柜，但要把门卫长期派上岗之前，必须确认柜子真能打开。

🗝

SecretRef 托管

像真正门卡不贴墙上，而是锁在安全柜里。

🧪

安装前先核验

向导会先确认那张取卡单真的能取到卡，不会傻乎乎直接开工。

🚫

拿不到就拦住

官方明确会阻止 daemon 安装，并给出补救提示，不会假装已经成功。

⚠️

token 和 password 打架

如果两种门禁都摆上了，却没说到底听谁的，向导也会停下来让你先做决定。

通道那间房

📮 名字、ID、allowlist 这一堆，其实就是“让门卫认得人”

Slack、Discord、Matrix、Teams 这些区块，官方会提示你填 allowlist，而且能尽量把名字解析成正式 ID。

🏷 先写昵称也行

像你告诉门卫“我想放行的是客厅群、运维群、小明的私聊”。

🔄 能翻成 ID 就帮你翻

向导会尽量把这些人话，变成系统真正认得的号码。

📜 allowlist

就是一张放行名单。没在名单上的门和房间，不随便让消息跑进来。

📌 一句话

这一步不是“填表”，而是在教门卫以后该认谁、放谁进。

最后总结

🎈 把这页压成一句最短的话

openclaw configure 就是带路老师模式：你说今天修哪几间房，它就边问边配，帮你定朝向、摆脑子、装侦察室、核验钥匙、整理门卫名单。

如果你下一步想自己动手精修某一格，就去 /cli/config。那页是同一栋楼里的“拿镊子版”。
