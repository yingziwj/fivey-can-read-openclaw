先讲人话版

⌨️ TUI 就像“把 OpenClaw 总控室搬进你的终端里”

如果 Dashboard 是浏览器里的总控室，那 TUI 就像把其中最常用的一套控制台塞进终端。你不用开网页，也能在命令行里聊天、切 agent、换 session、看工具卡片、调 thinking 和 delivery。

所以这页不是“又一个聊天方式”那么简单，它更像一块给终端用户准备的控制面板。

适合喜欢终端、想尽量不离开 shell 的人 这页最重要的是先看懂：TUI 连的是 Gateway，不是本地单机聊天小窗 查看官方原文

第一站

🖥 先想画面：你坐在终端里，但其实连着总控室

官方 quick start 第一条就先让你启动 Gateway，这不是废话。它是在提醒你：TUI 本质上是 Gateway 的一个前台，不是脱离网关独立活着的聊天器。

openclaw gateway

像先把整栋楼的电闸打开。TUI 只是前台屏幕，屏幕后面的楼得先有电。

openclaw tui

像在终端里打开这块前台屏幕。

--url ... --token ...

像你不是坐在本楼前台，而是远程接到另一栋楼的屏幕上，所以要带着地址和门票过去。

⭐ 一句话

TUI 是终端里的控制台，但背后仍然是 Gateway 在运转。

第二站

🧭 这块屏幕上你到底会看到什么

官方列 Header、Chat log、Status line、Footer、Input。翻成人话，就是一张结构很清楚的驾驶舱。

🪟

Header

像挡风玻璃上方那排状态：你连的是哪、现在是谁、在哪个会话里。

💬

Chat log

像主视野，用户话、回复、系统提示、工具卡片都在这里流过。

📶

Status / Footer

像仪表盘，告诉你连接状态、模型、thinking、tokens、deliver 开没开。

⌨️

Input

像驾驶位上的操纵杆。你写消息、敲命令、补 directive，都是从这儿出手。

第三站

🧠 Agents + Sessions 这段，其实是在讲“你现在坐在哪个工位、翻的是哪个抽屉”

很多人第一次看 session key 会头大。其实你只要记住：agent 像工位，session 像这个工位里的具体抽屉。

🧑 agent

像不同的工位，比如 main、research、design。每个工位负责不同风格的活。

🗂 session

像工位里的抽屉。同一个 agent 可以有很多抽屉，每个抽屉装一条任务线。

agent:<agentId>:<sessionKey>

像完整抽屉标签：哪张桌子、哪个抽屉，一次写清楚。

📌 一句话

切 agent 像换桌子，切 session 像换抽屉。

第四站

🕹 slash commands 和快捷键，像驾驶舱上的按钮

这页里按钮很多，但其实都能按“切视图、切档位、切对象、停当前动作”来记。

/agent / /session / /model

像换桌子、换抽屉、换脑子，是最常见的切换按钮。

/think / /fast / /verbose

像调当前驾驶模式，决定这次跑得快一点、想得深一点、还是把后台噪音多露出来一点。

Ctrl+L / Ctrl+G / Ctrl+P

像直接弹出模型、agent、session 选择器，不用每次手敲完整命令。

Esc / /abort

像踩刹车，告诉当前这次跑得太久了，先停。

第五站

📦 Tool cards 和本地 ! 命令，是两套完全不同的能力

这部分很容易混。一个是 Gateway 那边跑出来的工具卡片，一个是 TUI 自己本地 shell。

🧰 Tool cards

像你看见后台某个工具正在干活、干了什么、结果如何。它属于 Gateway 那边的正式工具事件。

! 本地 shell

像你在 TUI 这台终端本机上临时开一个小外壳，跑一条本地命令。它不是把命令发给 Gateway 去远端执行。

🔐 为什么本地 ! 要先提示一次

因为这已经不是聊天，而是在你当前终端主机上直接动手。系统会先确认你真的愿意开这扇门。

🎈 一句话

Tool cards 是看远端工具干活，! 是在眼前这台机器本地开 shell。

第六站

🧯 Troubleshooting 这一段其实很朴素：没反应时，先看是不是根本没连上

官方最后给的排查顺序很务实，而且跟这整站的风格一样，应该先做基础检查。

/status

像先看驾驶舱仪表盘：现在是连着、忙着、还是根本掉线了。

openclaw logs --follow

像去后面看监控，确认消息有没有真的进楼、工具有没有跑起来。

--url 配了就别指望自动兜底

官方特别提醒：你一旦手动指定远端 URL，就别再指望系统自动从配置里帮你补 token / password。门牌你自己填了，门票也得你自己带齐。

🎈 这节一句话

TUI 不出字时，先查连接和认证，别先怀疑模型人格突然变了。

最后总结

🎈 把 TUI 这页压成一句话

TUI 就是终端里的 OpenClaw 控制台：你在 shell 里切 agent、切 session、看工具卡片、调思考档位，但它背后连的仍然是 Gateway 这栋楼。

如果你下一页只想继续看一页，我建议回头看 /web/dashboard 或继续看 /web/webchat。前者是浏览器里的总控室，后者是另一种“看起来像聊天，其实也直连 Gateway”的前台。
