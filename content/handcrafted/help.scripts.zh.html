先讲人话版

📜 Scripts，就是仓库里那群“常用小按钮”

这页讲的是项目里常跑的脚本。你不用每次都自己拼长命令，按钮已经帮你收好了。

像厨房里贴好的快捷键，煮水、开火、关火都有现成标签。

适合想知道 npm run 这些按钮干什么 重点是 build、sync、check、verify-live 查看官方原文

第一站

🎛 脚本像遥控器，不像魔法

每个脚本都是把一串容易忘的动作收成一个按钮。

🏗

`npm run build`

把站点重新搭起来。

🔄

`npm run sync`

把原站内容重新抓一遍。

✅

`npm run check`

做一次生成结果体检。

第二站

🛠 脚本像在做什么

官方把脚本整理出来，是为了让你少记命令，多点按钮。

`build`

像把整栋房子重新打扫一遍。

`verify-live`

像到门口看线上房子亮没亮灯。

`sync`

像回原仓库拿最新材料。

最后记住

🎈 Scripts 的价值，就是让重复动作按一下就走

把常用动作收成脚本，站点维护就会轻很多。
