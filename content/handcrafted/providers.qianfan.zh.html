先讲人话版

🐼 Qianfan 像“百度云里的模型总柜台”

这页非常短，因为主线也非常短。Qianfan 不是一颗单独模型，而是百度云里的统一模型柜台。你先去开账户、领 API key，再让 OpenClaw 走这条总柜台路就行。

适合想接百度千帆的人重点是云账号、统一入口、专属 API key查看官方原文

第一步

🪪 先把“百度云账号 + 千帆 API 权限 + API key”凑齐

官方把前置条件列得很清楚。翻成人话，就是你得先拿到进这个柜台的资格和钥匙。

☁️ Baidu Cloud 账号

像先办一张能进这栋云大楼的总门卡。

🧾 Qianfan API access

像在楼里再开通“模型柜台”这层权限。

🔑 bce-v3/... 形式 API key

像千帆柜台给你的专属开门钥匙。

第二步

🛠 openclaw onboard --auth-choice qianfan-api-key 像“在向导里说我要走百度这扇门”

这条命令没有隐藏戏法。它就是把你从 OpenClaw 的大门带到 Qianfan 这扇分门前，然后收下钥匙。

所以这页真正的重点不是命令本身，而是你得先去千帆控制台把钥匙办出来。没有钥匙，向导只是一个空登记台。
