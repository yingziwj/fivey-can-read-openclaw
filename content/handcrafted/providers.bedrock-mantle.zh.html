先讲人话版

🪨 Bedrock Mantle 像 AWS 云港口里的一扇模型门

它让你用 AWS 的凭证体系去拿一个 Mantle 的 OpenAI 兼容入口，再去碰 GPT-OSS、Qwen、Kimi、GLM 这类模型。

这页最难的不是模型，而是你到底用钥匙进，还是让 AWS 凭证链帮你生一把临时门卡。

适合 AWS 用户重点是 bearer token、region 和自动发现查看官方原文

第一步

🔐 先选门卡来源

你可以直接给 bearer token，也可以让 AWS 默认凭证链帮你出卡。

AWS_BEARER_TOKEN_BEDROCK

像你自己手里已经有门卡，直接刷。

AWS 凭证链

像系统帮你临时发一张卡，再去查模型目录。

AWS_REGION

像告诉港口你现在在哪个海湾靠岸。

第二步

🧭 Mantle / Bedrock 像“云港口里的中转柜台”

它不是单一模型，而是给你一堆托管模型的统一门口。

🧠

自动发现

像先把目录牌翻出来，看今天港口里有哪些老师在值班。

🌍

支持多个 region

像同一港口有好几个海湾入口。

📌

OpenAI 兼容

像给港口配一个大家都看得懂的窗口格式。

最后总结

🎈 把 Bedrock Mantle 这页压成一句最好记的话

Bedrock Mantle 是 AWS 云港口里的模型柜台。你先选门卡，再选海湾，系统就能帮你把 GPT-OSS、Qwen、Kimi、GLM 这类模型接进来。
