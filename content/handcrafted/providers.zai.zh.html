先讲人话版

🧪 Z.AI 这页，其实就是 GLM 的“官方卖票处”

如果说 GLM 是模型家族名字，那 Z.AI 就像这家模型的官方售票平台。你去 Z.AI 办 API key，再让 OpenClaw 以 zai/... 这种前缀去叫模型。

适合想正式接 GLM 家族的人重点是四个 auth choice、GLM 模型前缀、tool_stream查看官方原文

第一步

🛣 四条 onboarding 命令，其实还是“地区 + 计划类型”四个窗口

跟阿里那页有点像，你先别背名字，先懂分类。

🌍 zai-coding-global

像国际区 Coding Plan 窗口。

🇨🇳 zai-coding-cn

像中国区 Coding Plan 窗口。

🧾 zai-global / zai-cn

像国际区和中国区的普通 API 窗口。

🎈 一句话

别死记命令，记住它们是四个售票窗口。

第二步

🧠 zai/glm-5 这种名字，像“售票平台 / 具体车型号”

官方给的默认例子非常简单，就是告诉你模型前缀怎么写。

你可以把 zai 想成售票平台名字，把 glm-5 想成具体车次。以后要换 GLM 版本，本质上就是换后面那节车厢名。

第三步

🌊 tool_stream 默认开着，像“工具回执会边走边播报”

官方特意提这句，说明它是这家 provider 的一个风格特征。

tool_stream

像工具动作不是等全部做完才汇报，而是边执行边往回播报进展。

关掉它意味着什么

像你不想听现场播报，只想等最后总结果，那就把这个开关关掉。

最后总结

🎈 把 Z.AI 这页压成一句最好记的话

Z.AI 是 GLM 的官方卖票处。先选对地区和计划窗口，再用 zai/模型名 去叫老师；如果不想边跑边听工具播报，再把 tool_stream 关掉。
