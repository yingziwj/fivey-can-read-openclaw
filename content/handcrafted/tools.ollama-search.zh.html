先讲人话版

🔎 Ollama Web Search 像给本地 Ollama 再装一只“搜网页的小眼睛”

它不是搜索引擎大厂，而是 Ollama 自己的实验搜索接口。你先让 Ollama 登录，再让 OpenClaw 去选它。

适合已经在用 Ollama 的人重点是 ollama signin 和 web search 配置查看官方原文

第一步

🪪 先让 Ollama 自己签到

没有 `ollama signin`，它就像没拿到进图书馆的门卡。

🏠

本地 Ollama

像你家里的一台会查资料的小机房。

🔐

`ollama signin`

像先登记账户，告诉它你是谁。

🧭

`openclaw configure --section web`

像把“网页搜索”这间房间门打开，然后选 Ollama。

第二步

🧰 它会复用你原来那台 Ollama 机器

如果你本来就有 Ollama 模型，它会直接借同一台主机，不用另起一套搜索服务。

无 key 默认

像不再额外办一张搜索卡。

自带 host

像直接沿用你原来那台机器。

可复用 auth

像模型和搜索共用同一把钥匙。

最后总结

🎈 把 Ollama Search 这页压成一句最好记的话

Ollama Web Search 是本地 Ollama 的网页眼睛。先签到，再选它当搜索供应商，就能把同一台本地机房的能力顺手借出来。
