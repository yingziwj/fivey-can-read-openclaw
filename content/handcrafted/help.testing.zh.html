第二站

🛠 命令像在做什么

测试命令像一套验收流程，从桌面到线上，逐层过关。

`npm run build`

像先把积木完整搭起来。

`npm run check`

像确认每块积木有没有插歪。

`npm run verify-live`

像去真门口看看灯是不是亮着。

最后记住

🎈 Testing 的目标很朴素：别让看起来能跑的东西，真跑时掉链子

把这套检查做稳，站点更新才更安心。
