先讲人话版

🛰 Chutes 像一个能用浏览器登录，也能直接拿 API key 的模型集市

它提供 OpenAI 兼容接口，OpenClaw 既能走 OAuth，也能走 key。官方还会在拿到认证后帮你发现模型。

这页的重点是：你是走浏览器登录，还是直接用 key。

适合想用 Chutes 模型目录的人 重点是 OAuth / API key 双路线 查看官方原文

第一步

🔐 两条入口，像两种进门方式

你可以走浏览器认证，也可以直接丢 key。

🌐

OAuth

openclaw onboard --auth-choice chutes

🪪

API key

openclaw onboard --auth-choice chutes-api-key

🧭

登录成功后会自动发现目录

像前台帮你抄今天的柜台名单。

第二步

🏷 默认模型和别名像“先给你安排一个常用老师”

Chutes 会先给一个默认模型，也顺手给几个便利别名。

chutes/zai-org/GLM-4.7-TEE

默认老师，通常会先上场。

chutes-fast

像快跑别名，指向更快的 GLM 版本。

chutes-pro

像更稳更强的备选别名。

第三步

📦 发现失败时，也要能兜底

官方很实际：如果发现不顺，还是会回落到内置目录，别让启动直接断。

Discovery

认证到位后，OpenClaw 会去 Chutes 目录看当前可用模型。

Fallback catalog

如果目录抓不到，也还有内置清单顶着。

适合场景

想要稳定目录，又不想自己手写一堆模型条目的人。

最后总结

🎈 把 Chutes 这页压成一句最好记的话

Chutes 是能走浏览器登录也能走 key 的模型集市：先认证，再发现目录，最后给你默认老师和便利别名。
