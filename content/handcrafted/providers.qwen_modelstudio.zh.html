先讲人话版

🏬 Model Studio 像“阿里云模型商场”，而且分普通售票口和订阅会员通道

这页最值得看懂的，不是某个具体模型，而是两种付费路线和四个入口地址。你要先知道自己走的是按量付费，还是订阅 Coding Plan。

适合接 Qwen / 阿里云模型的人重点是 Standard 与 Coding Plan、国内外入口查看官方原文

第一步

🛣 先分两条收费路：Standard 和 Coding Plan

官方把这件事摆得很前。因为你如果一开始路都站错了，后面 endpoint 也会跟着错。

💳

Standard

像普通买票进商场，按用量结账。

🎟

Coding Plan

像会员包月通道，更偏订阅路线。

🌏

Global / China

像同一家商场有国际门和中国区门，进哪扇门决定你走哪个入口地址。

🎈

一句话

先确认付费路线，再确认地区入口。

第二步

🛠 这四条 onboarding 命令，像“四个售票窗口”

别被命令名吓到，本质上就是四个窗口：标准版中国、标准版国际、订阅版中国、订阅版国际。

modelstudio-standard-api-key-cn

像中国区按量付费窗口。

modelstudio-standard-api-key

像国际区按量付费窗口。

modelstudio-api-key-cn / modelstudio-api-key

像中国区和国际区的订阅会员窗口。

🎈 最短记法

四个窗口，区别只有“地区 + 付费方式”。

第三步

🧠 Model Studio 不只卖 Qwen，它像“一个混合品牌商场”

官方特意列出 GLM、Kimi、MiniMax，不是在跑题，而是在提醒你：这家商场里不只摆 Qwen 自家货。

所以 modelstudio/qwen3.5-plus 只是一个常见默认货架。你也可以在同一个商场里逛到 GLM、Kimi、MiniMax 这些别家的货。

最后总结

🎈 把 Model Studio 这页压成一句最好记的话

Model Studio 是阿里云模型商场。先站对售票窗口，再选默认货架，别把“Qwen 商场”误会成“只卖 Qwen”。
