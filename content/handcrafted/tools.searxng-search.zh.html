先讲人话版

🕸 SearXNG Search 像给 OpenClaw 装一个“自己家里开的搜索超市”

它是自托管、不要 key 的 web_search provider。也就是说，查询不会跑去商业搜索 API 那边，而是走你自己的 SearXNG 实例。

这页特别适合在意隐私、预算，或者想完全自己控搜索结果的人。

适合自托管搜索的人 重点是 docker run、SEARXNG_BASE_URL、https 限制 查看官方原文

第一步

🏗 先起一个 SearXNG 小站

最直白的起手式就是自己跑一台实例。

1

docker run ... searxng/searxng

像先把搜索超市开起来。

2

openclaw configure --section web

像把网页搜索门牌改成 SearXNG。

3

SEARXNG_BASE_URL

像告诉 OpenClaw 这家超市在哪。

第二步

🔒 私有、免费、可控，是它的三大卖点

官方把优势写得很实在。

Free and unlimited

不需要商业搜索 key。

Privacy

请求只在你自己的网络里走。

Works anywhere

没有地域商业限制。

第三步

📦 配置时要注意 https 和 baseUrl

这类自托管搜索，最容易出错的是地址和协议。

provider: "searxng"

像把搜索按钮指向 SearXNG。

baseUrl

像写清楚这家超市的门牌号。

https 规则

公开 host 要用 https，http 只给可信内网或 loopback。

最后总结

🎈 把 SearXNG 这页压成一句最好记的话

SearXNG 就是自家开的搜索超市：自己起实例、自己配门牌、自己控制隐私和结果来源。
